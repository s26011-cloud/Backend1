# DBMS
- DBMS는 데이터베이스를 체계적으로 관리하여, 다양한 유형의 데이터를 효율적으로 처리하는데 필수적인 소프트웨어.

## 대표적인 기능

- 정의:
데이터에 대한 형식, 구조, 조건들을 정의하는 기능이다.

- 저장:
기억장치에 데이터를 저장하는 기능이다.

- 보안:
하드웨어나 소프트웨어의 오류 또는 권한이 없는 접근으로부터 시스템을 보호한다.

- 공유:
여러 사용자와 프로그램이 데이터베이스에 접근을 할수 있도록 공유한다.

- 기능:
데이터의 검색을 위한 질의나 데이터베이스의 갱신, 생성기능을 포함한다.

- 유지:
요구사항의 변화에 다라 반영할 수 있도록 하는 기능이다.

## 관계형 DBMS
데이터를 테이블 형식을 저장하며,SQL을 사용하여 데이터에 접근하고 관리한다.

### 특징

- 데이터의 테이블 기반 구조

- 데이터 무결성 및 일관성 보장

- SQL 사용

## 비관계형 DBMS(NoSQL DBMS)
비관계형 DBMS는 관계형 모델과 달리 비정형 또는 반정형 데이터를 처리하는데 최적화된 데이터베이스 모델.

### 특징

- 유연한 스키마

- 다양한 데이터 형식 지원
