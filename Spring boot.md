# Spring Boot란?
Spring Boot는 Java Spring Framework를 사용하여 웹 애플리케이션 및 마이크로서비스를 더 빠르고 쉽게 개발할 수 있는 도구이다.

## 핵심 기능

- 자동 구성

- 구성에 대한 독단적인 접근 방식

- 독립형 애플리케이션 생성 가능

### 자동 구성
자동 구성은 사전 설정된 종속성으로 애플리케이션을 초기화하므로 수동으로 구성할 필요가 없다.

### 독단적 접근 방식
사용자가 직접 모든 결정을 내리고 모든 것을 수동으로 설정하지 않아도 자체 판단에 따라 설치할 패키지와 사용할 기본값을 선택한다.

### 독립형 애플리케이션
다른 소프트웨어나 서버의 도움 없이 단일 컴퓨터 시스템에서 직접 설치 및 실행되는 독립적인 프로그램이다.
